# Suggested commands
- Run all tests: `swift test`
- Build app/package: `swift build`
- Run the macOS app from SwiftPM: `swift run BullpenApp`
- Inspect files quickly: `rg --files`, `rg <pattern> Sources Tests`
- Standard macOS shell tools available: `git`, `ls`, `cd`, `find`, `sed`, `awk`, `xargs`
